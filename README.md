#Halfsies
## Indent Based HTML Without Closing Tags

Halfsies turns this

```
<div>Hi
  <h1>Ba
  <p>
     Line one is a really long line
     Line two makes an <b>important point
```

into this
```
<div>Hi
  <h1>Ba</h1>
  <p>
     Line one is a really long line
     Line two makes an <b>important point</b>
  </p>
</div>
```
